AWS Observability + PostgreSQL DBRE Platform

=============================================
=============================================

Overview
--------
This project demonstrates real-world Cloud Engineering, Database Reliability Engineering (DBRE), and Site Reliability Engineering (SRE) practices using AWS, Terraform, Docker, PostgreSQL, GitHub Actions, and operational monitoring workflows.

The objective was to build a production-style observability and database platform that showcases infrastructure-as-code, monitoring, alerting, backup/recovery, container operations, and operational ownership.


Skills Demonstrated
-------------------
	Cloud Engineering:
	
	- Infrastructure as Code
	- AWS monitoring and observability
	- Alerting strategy
	- Cloud auditing
	- Resource tagging standards


	Database Reliability Engineering:
	
	- PostgreSQL administration
	- Backup automation
	- Recovery validation
	- Database platform operations
	- Operational troubleshooting

	Site Reliability Engineering:
	
	- Monitoring and alerting
	- Operational ownership
	- Incident response concepts
	- Reliability validation
